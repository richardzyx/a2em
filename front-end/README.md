Welcome to the front end server for A2Empowerment.

Installation requirements:

1. Install Git

2. Install Node and NPM

3. Download the code

4. npm install -g requirements.txt

5. Run: DEBUG=myapp npm start (OS X/Linux)

6. Visit http://localhost:3000/
